AIFLASH — AI Flashcards & Study Assistant

Overview
--------
Next.js application that generates study flashcards and practice questions using an LLM. Includes authentication and optional Firestore persistence.

Features
--------
- Generate flashcards from text or topics
- Session progress and export options
- User auth via Firebase

Tech stack
----------
Next.js, Tailwind CSS, Firebase (Auth/Firestore), OpenAI

Quick start
-----------
1. npm install
2. create .env.local with NEXT_PUBLIC_FIREBASE_API_KEY and OPENAI_API_KEY
3. npm run dev

Notes
-----
See `firebase.ts` and `middleware.ts` for env var requirements.
